## Professional (Personal) Portfolio

## Overview
Created and maintained a personal portfolio demonstrating abilities with responsive grid structures, interactive hover states, custom typography, and reusable UI elements across pages. Designed with structured information hierarchy and subtle visual styling to highlight content while maintaining a professional aesthetic. 
Accessible through the following link: https://veziok-portfolio.netlify.app/


## Usage
### File Responsibilities 
#### index.html
- Serves as the landing page and primary introduction
- Provides global navigation links to all portfolio sections

#### experience.html
- Uses reusable experience card components for consistency
- Maintains shared navigation and footer layout with other pages

#### projects.html
- Showcases technical projects and academic research in a unified layout
- Uses reusable project card components with external GitHub and PDF links

#### skill_award.html
- Presents categorized technical, analytical, and professional skills
- Organizes skills into structured groups for scannability and clarity

#### style.css
- Defines the global visual identity using custom typography, spacing, and color palette
- Implements responsive layouts with multiple breakpoint-based media queries
- Uses Flexbox and CSS Grid for structured, scalable page layouts
- Provides consistent styling for navigation, hero sections, cards, and footers
- Includes accessibility utilities (e.g., visually-hidden class) and smooth scrolling
- Handles hover states, transitions, and subtle motion for interactive feedback
